# TBOI - Asset Compiler For Photoshop

Herramienta local para generar un script `.jsx` que importa archivos `.png` de forma masiva en Photoshop como capas rasterizadas individuales, conservando su nombre original y evitando el suavizado o deformación por defecto de Photoshop.

## 📋 Requisitos
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)
- Adobe Photoshop

## 🛠️ Instalación
En la raíz del proyecto, instala las dependencias con:
```bash
pnpm install
